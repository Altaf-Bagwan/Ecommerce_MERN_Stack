## Tech Stack
The project has been built using the following technologies:

- Frontend:

  - React
  - Material UI
  - HTML, CSS, JavaScript

- Backend:

  - Node.js
  - Express.js
  - REST API

- Database:

  - MongoDB on Atlas Cloud
